# 一、Java常见算法
## 1.排序算法
### 冒泡排序

```     //Bubble Sort
        int length = arr.length;
        boolean flag;
        for (int i = length - 1; i > 0; i--) {
            flag = true;
            for (int j = 0; j < i; j++) {
                if (arr[j] > arr[j + 1]) {
                    arr[j] = arr[j + 1] + arr[j];
                    arr[j + 1] = arr[j] - arr[j + 1];
                    arr[j] = arr[j] - arr[j + 1];
                    flag = false;
                }
            }
            if (flag) {
                break;
            }
        }
```


# 梳排序

```
        //Comb Sort
        int length = arr.length;
        int gap = length;
        float shrink = 1.3f;
        boolean swapped;
        do {
            swapped = false;
            gap = (int) Math.floor(gap / shrink);
            if (gap < 1) {
                gap = 1;
            }
            for (int i = 0; i + gap < length; i++) {
                if (arr[i] > arr[i + gap]) {
                    arr[i] = arr[i] + arr[i + gap];
                    arr[i + gap] = arr[i] - arr[i + gap];
                    arr[i] = arr[i] - arr[i + gap];
                    swapped = true;
                }
            }
        } while (gap != 1 || swapped);
```
